12 modèles d’email et SMS pour relancer un devis sans réponse.

Des messages courts, professionnels et faciles à adapter pour artisans, entreprises du bâtiment, rénovation, climatisation et services locaux.

Une bonne relance ne met pas la pression. Elle rappelle le devis, facilite la réponse et donne au client une prochaine étape claire.

Le plus important est la régularité : une relance à J+2, une autre quelques jours plus tard, puis un dernier message propre si le client ne répond pas.

Rythme conseillé

  • J+2 : vérifier que le devis a bien été reçu
  • J+7 : répondre aux questions ou ajuster le projet
  • J+14 : demander une décision ou un retour clair
  • J+21 : clôturer proprement sans fermer la porte
Emails prêts à adapter

Modèles d’emails de relance devis.

1. Première relance douce

Objet : Votre devis pour [projet]

Bonjour [Nom], je reviens vers vous concernant le devis envoyé pour [projet]. L’avez-vous bien reçu ? Je reste disponible si vous avez une question ou besoin d’une précision.

2. Relance après quelques jours

Objet : Suite à votre demande de devis

Bonjour [Nom], avez-vous eu le temps de regarder notre proposition ? Si vous souhaitez avancer, je peux vous confirmer les prochaines disponibilités.

3. Relance avec disponibilité

Objet : Créneau possible pour [projet]

Bonjour [Nom], je peux encore prévoir un créneau pour votre chantier sur [période]. Souhaitez-vous que l’on bloque une date ou préférez-vous ajuster certains points du devis ?

4. Relance rénovation

Objet : Devis travaux [pièce / logement]

Bonjour [Nom], je me permets de revenir vers vous pour votre projet de rénovation. Si le budget ou le planning doit être ajusté, dites-moi ce qui bloque et je vous réponds rapidement.

5. Relance climatisation

Objet : Installation climatisation

Bonjour [Nom], je reviens vers vous pour le devis climatisation. Si vous souhaitez être installé avant [période], il vaut mieux valider les prochaines étapes rapidement.

6. Relance après appel

Objet : Suite à notre échange

Bonjour [Nom], suite à notre échange, je vous confirme que je reste disponible pour finaliser le devis ou répondre à vos dernières questions avant décision.

7. Relance objection prix

Objet : Ajustement possible du devis

Bonjour [Nom], si le budget est le point principal, je peux regarder avec vous les options possibles pour adapter la proposition sans dégrader le résultat attendu.

8. Relance décision

Objet : Souhaitez-vous donner suite ?

Bonjour [Nom], je voulais savoir si vous souhaitiez donner suite au devis ou si vous préférez que je clôture la demande pour le moment. Dans les deux cas, votre retour m’aide à m’organiser.

9. Dernière relance propre

Objet : Dernier retour sur votre devis

Bonjour [Nom], sauf erreur de ma part, je n’ai pas eu de retour sur le devis. Je clôture le suivi pour l’instant, mais je reste disponible si le projet redevient d’actualité.

10. Relance ancien devis

Objet : Votre projet est-il toujours prévu ?

Bonjour [Nom], nous avions échangé au sujet de [projet]. Le besoin est-il toujours d’actualité ? Si oui, je peux vous confirmer les délais et mettre à jour le devis si nécessaire.

11. SMS court

Bonjour [Nom], c’est [Entreprise]. Avez-vous eu le temps de regarder le devis pour [projet] ? Je reste disponible si besoin. Bonne journée.

12. SMS décision

Bonjour [Nom], souhaitez-vous que l’on avance sur le devis [projet], ou dois-je clôturer la demande pour le moment ? Merci pour votre retour.
